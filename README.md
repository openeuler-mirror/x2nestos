# x2nestos

将通用形态操作系统转换为 NestOS 的快捷部署工具。

#### 介绍

基于已部署的 openeuler 等通用 OS 形态，转换为基于不可变操作系统的 NestOS。适用于不方便重新引导安装或大批量转换的场景。

**注意：**

**1.非迁移工具，不保留磁盘数据。**

**2.此转换不可逆。**
